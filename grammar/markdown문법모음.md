# 마크다운(markdown) 코드블록(code block) 사용법과 문법 강조
마크다운에서는 백쿼트(`)를 사용하여 소스 코드를 쉽게 표시하거나 강조할 수 있음

## 인라인으로 사용
문장 사이에 백쿼트(`) 한개를 사용해 입력하면 인라인으로 소스 코드를 표시할 수 있음

예) 소스 코드는 `var test = 123` 입니다.

## 코드 블록으로 사용
백쿼트(`) 세개를 사용하여 코드 블록을 사용할 수 있음

 ```

 ```
